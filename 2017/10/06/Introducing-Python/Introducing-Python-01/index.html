4. 파이썬을 쓰면 안 될 때

파이썬은 모든 상황에서 좋지만 프로그램이 계산 작업을 많이 하는 경우에는 부적합하다

6. 파이썬 설치하기

컴퓨터에 Python이 설치가 안되어있다는 가정하에 python을 설치 한다.
https://www.python.org/downloads/에서 3.5.x 버전을 다운받아서 설치한다.
많은 버전이 있지만 나는 3.5.x버전이 제일 편하더라
